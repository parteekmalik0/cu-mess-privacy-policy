Privacy Policy for Cu Mess

Last updated: July 9, 2025

Cu Mess respects your privacy. This Privacy Policy explains how information is handled in the app.

1. Information We Collect

Cu Mess does not require users to create an account or provide personal information.
However, the app uses certain services that may process limited data necessary for functionality, including:
App interaction data (used to fetch and display meal information from Firebase Firestore)
Device identifiers (used for push notifications via Firebase Cloud Messaging)
No sensitive personal information such as name, email, phone number, location, financial information, or contacts is collected.

2. How We Use Information

Information is used only to:
Display meal-related content
Send push notifications
Maintain and improve core app functionality

3. Third-Party Services

Cu Mess uses Google Firebase services, including:
Firebase Firestore (cloud database)
Firebase Cloud Messaging (push notifications)


4. Data Security

All data transmitted between the app and Firebase servers is encrypted using secure HTTPS protocols.

5. Data Deletion

Since Cu Mess does not require user accounts, users may stop data collection by uninstalling the app.
